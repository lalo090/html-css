Lista Descriptiva

Funcion
Para devolver un valor, la función debe tener una sentencia return que especifica el valor a devolver.

Let
Let te permite declarar variables limitando su alcance al bloque, declaración, o expresión donde se está usando.
Otra diferencia entre var y let es que este último se inicializa a un valor sólo cuando un analizador lo evalúa.

Const
Las variables constantes presentan un ámbito de bloque (block scope) tal y como lo hacen las variables
definidas usando la instrucción let, con la particularidad de que el valor de una constante no puede cambiarse
a través de la reasignación. Las constantes no se pueden redeclarar.

Meta
Etiqueta usada para definir otros metadatos que no se pueden definir con una etiqueta HTML especifica. Por ejemplo para definir el autor del sitio, o la descripción del mismo.
Footer
Usadas para definir el pie de página.
Hr
Etiqueta utilizada para «romper» entre dos secciones de una web. Usada comúnmente como separador.
